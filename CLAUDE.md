# Project: LUMEA Landing

## Objective
Create a high-converting landing page to sell SEO services.

## Target
- Business owners
- Local companies
- People who want more clients from Google

## Priorities
1. Conversion (MOST IMPORTANT)
2. SEO optimization
3. Clear messaging

## Rules
- Every section must drive conversion
- Avoid generic text
- Focus on results (more leads, more clients)
- Use simple language

## Skills Usage
- Use landing-builder for structure
- Use copywriting for messaging
- Use seo-audit to optimize

## Behavior
Before generating content:
1. Think about user intent
2. Identify pain points
3. Propose structure
4. Then write

## Tone
- Professional but simple
- Results-driven
- No fluff
